## Caso 002 – Ingreso y validación de datos en Python

## Escenario
Script que permite ingresar números por teclado, validar entradas, almacenar valores y mostrar un resumen final.

## Objetivo
Practicar validación de datos, bucles, listas y manejo de errores.

## Herramientas usadas

input()

while

listas

try/except

funciones básicas (sum, len)

## Resultado
El programa valida correctamente las entradas, evita errores por datos inválidos y muestra estadísticas básicas al finalizar.

## Aprendizajes

Cómo validar entradas sin romper el programa

Uso práctico de listas

Importancia del manejo de errores
