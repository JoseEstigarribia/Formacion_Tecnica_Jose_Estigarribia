## Ingreso y validación de números enteros

### Descripción
Script en Python que permite ingresar números por teclado, validarlos como enteros, almacenarlos en una lista y mostrar un resumen final con cantidad, suma y promedio.

### Funcionalidades
- Validación de entrada con try/except
- Uso de listas
- Separación de lógica en funciones
- Cálculo de estadísticas básicas

### Aprendizajes
- Manejo de errores
- Estructuración de código
- Uso de funciones y main()
