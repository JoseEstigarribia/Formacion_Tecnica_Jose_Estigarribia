
# ¿Qué es una API?

Una API (Application Programming Interface) es un medio que permite que dos sistemas se comuniquen entre sí.

Las APIs permiten:

* Enviar datos
* Recibir información
* Ejecutar acciones en otro sistema

Ejemplo:
Un sistema puede enviar datos a Google Sheets o recibir información desde una plataforma externa utilizando una API.

En automatización, las APIs son fundamentales porque permiten conectar distintos servicios sin intervención manual.
